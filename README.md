# 🌦️ Skywatcher — Weather App

**Skywatcher** is a modern weather web application that shows real-time weather conditions and a 5-day forecast for any city worldwide.
It features GPS-based location detection, dark/light theme switching, search history, and works without requiring any API key.

---

## 🚀 Features

- 🔍 Search weather by city name
- 📍 Detect weather using GPS location
- 📅 5-day forecast display
- 🌙 Dark / ☀️ Light theme toggle
- 🕘 Recent search history support
- 📱 Fully responsive design
- ⚡ Fast performance
- 🔐 No API key required

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- wttr.in weather service

## 🌍 How It Works

Skywatcher fetches weather data using:

https://wttr.in

This service provides free weather information in JSON format without authentication.

## 🚀 Deploy Using GitHub Pages

1. Go to Repository **Settings**
2. Click **Pages**
3. Select **Deploy from branch**
4. Choose **main → /root**

Your site will be live at:
